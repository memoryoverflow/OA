### 使用技术

#### 服务端： springboot(2.2.1) + mybatis + shiro(1.4.0)  + redis + activiti(5.2.1)

#### pc端：vue2.0 + element-ui


<br>
<br>

###  项目结构

- web_pc:新版本的pc端

- sys-oa: 新版本的服务端

- 旧的是：前后一体的
